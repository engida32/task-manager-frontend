.env content is

 -->
1. Clone the repository
```bash
git clone
```
2. Install the dependencies
```bash 
yarn install
```
3. Create a .env file in the root of the project and add the following content
```bash
REACT_APP_API_BASE_URL=
```
4. Start the project
```bash
yarn start
```
5. Open the browser and go to http://localhost:3000
```
